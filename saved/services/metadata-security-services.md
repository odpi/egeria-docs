# Metadata security services

The *metadata security services* supports the connector interfaces and the mechanisms to invoke any configured [metadata security connectors](/concepts/metadata-security-connectors).  It is implemented by the [metadata-security](https://github.com/odpi/egeria/tree/main/open-metadata-implementation/common-services/metadata-security) module.

The [Metadata Security](/features/metadata-security/overview) feature describes how the metadata security connectors operate.
