# Actor Profile

An *actor profile* represents a person, group of people, or an automated process operating in the open metadata ecosystem. It provides more details of the identity, capabilities an purpose behind a userId associated with a request.

??? info "Further information"
    * The [ActorProfile](/types/1/0110-Actors) open metadata type defines the properties and relationships for an actor profile.
