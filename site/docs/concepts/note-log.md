# Note log

A *note log* is an attachment to a [referenceable](/concepts/referencable) that acts as a series of ordered notes.  The entries in a note log are [actions](/concepts/action).  

???+ info "Additional information"
    * The type definitions for a note loge are found in [Model 0160](/types/1/0160-Notes).
