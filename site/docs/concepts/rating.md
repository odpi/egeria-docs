# Rating

A rating is an attachment that can be added to an element. It includes a star rating and an optional comment.

Ratings are used to provide feedback on ideas, projects, perceived quality of assets and digital products etc.

??? info "Further information"
   The *Rating* open metadata type is described in [Model 0150 Feedback](/types/1/0150-Feedback).
