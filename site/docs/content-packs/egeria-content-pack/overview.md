# Egeria Content Pack

The *EgeriaContentPack* contains the connector definitions for the Egeria infrastructure management connectors and governance services supplied in the *omag-server-platform* distribution along with the valid metadata values for the technologies they support. This content pack is designed to provide a good starting point to understand the structure and status of the Egeria ecosystem.

This content pack is built by the [core-content-pack](https://github.com/odpi/egeria/tree/main/open-metadata-resources/open-metadata-archives/) module.
