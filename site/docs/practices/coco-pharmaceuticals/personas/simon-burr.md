# Simon (Si) Burr - IT Security

## Persona Description

----

![Icon](simon-burr.png)

Age: 27

Skills: Cyber-security, penetration testing

Scope: IT - reviews and validates of IT systems and services.

Job:

* Si Burr is a recent hire, responsible for validating the security of Coco Pharmaceuticals' IT systems and services.
* Si officially works for [Ivor Padlock](/practices/coco-pharmaceuticals/personas/ivor-padlock) but spends most of his time working with [Polly Tasker](/practices/coco-pharmaceuticals/personas/polly-tasker) on the active IT projects and [Gary Geeke](/practices/coco-pharmaceuticals/personas/gary-geeke) on the over all IT infrastructure security.

Thinks/Feels:

* Si is very interested in security and is concerned that Coco Pharmaceuticals has no regular security scanning and penetration testing.
* Si feels a bit behind on the changes that AI is bringing to security - both in terms of new threats and capabilities - but is keen to learn more.

Hears:

* Excitement at the increased investment in Egeria, the data management platforms and AI.

Says/Does:

* Provides explanations and education sessions on how IT system security needs to be managed in the modern era.

Sees:

* Deep focus on research and new investments in AI.

Top challenges:

* IT System Security is a very complex topic and is changing rapidly.
* Coco is currently underinvested in IT System Security scanning and testing.
* Keeping up-to-date with the impact of AI on security.

Desired Outcome:

* He would like to set up automated security testing, scanning and monitoring of Coco's IT systems and services.
* He would like to change the software development approach to include a security-first perspective
