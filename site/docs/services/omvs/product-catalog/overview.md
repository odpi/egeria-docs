# Product Catalog API

The Product Catalog API provides a REST API to support user interfaces (UIs)
relating to the discovery and use of digital products.

## Key Features

The Product Catalog API supports the following key features:

* **Catalog Search**: Search and browse available digital product catalogs to discover digital products and product families.
* **Product Discovery**: Explore the structure and contents of digital products.
* **Product Subscription**: Request and manage subscriptions to digital products to receive and use their data.

* [Digital Subscription Concept](/concepts/digital-subscription/)
